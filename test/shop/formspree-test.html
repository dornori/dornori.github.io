05 — Integration Checklist

    
    
      
  • 
        1
        Add the Turnstile CDN script to every page that uses the checkout:
          
    <script src="https://challenges.cloudflare.com/turnstile/v0/api.js" defer></script>
            
      
    • 
      
  • 
        2
        Set your real Turnstile site key in config.js →
          CONFIG.turnstile.sitekey. Also add the secret key to your
          Formspree form settings (Formspree → Settings → Integrations → Cloudflare Turnstile).
        
      
    • 
      
  • 
        3
        In your Formspree form settings, enable the Turnstile integration and
          paste your secret key. Formspree will then verify every token
          server-side before accepting a submission.
        
      
    • 
      
  • 
        4
        The captcha container (#lumio-captcha-container) in
          cart.html renders automatically when the customer clicks
          "Continue to Payment". The widget must pass before the order is sent.
        
      
    • 
      
  • 
        5
        On localhost, fetch to Formspree may be blocked by CORS.
          Deploy to any static host (Netlify, Vercel, GitHub Pages) to test end-to-end.
