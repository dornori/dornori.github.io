Language & Currency

Language priority: ?lang=nl URL param → localStorage → CONFIG.language → "en" fallback. Switching language fires shop:langChanged — all components re-render without a page reload. Embedding sites (Dornori etc.) pass ?lang=nl in the link/iframe URL.

Live — switch language or currency above ↑

// Wire lang buttons (any selector)
Shop.wireLanguageSwitcher(".lumio-lang-btn");

// Currency selector dropdown
Shop.renderCurrencySelector("#currency-slot");

// Always boot in this order:
Promise.all([
  Shipping.load(),
  Currency.init(),   // IP detect + CSV
  Shop.loadLang(),
]).then(() => {
  Shop.wireLanguageSwitcher(".lumio-lang-btn");
  // ... render components
});

// Embed from Dornori with lang preset:
// lumio.shop/index.html?lang=nl

Product URL — Clickable Image

Images are not clickable by default. Set product.url to make the image an <a> link. Leave it empty and the image is a plain <div> — no dead cursor, no broken behavior.

attachBuyOverlay() has been retired. No modal, no overlay. Clean and predictable.

With url set — pointer cursor, link tag

Without url — no cursor, div tag

// JSON with URL → image is <a href="...">
{ "url": "https://lumio.shop/products/arc-floor-lamp" }

// JSON without URL → image is <div>
{ "url": "" }  // or omit the field entirely

// Rendered HTML:
<a href="..." class="lumio-card-img-link">   ← url set
<div class="lumio-card-img-wrap">           ← no url

Variant Pricing

Each product can have a variants array. Variants override price, image, weight, and stock — any field can be omitted and falls back to the base product value. 

{
  "id": "hoody",
  "price": 89,    ← base/fallback price
  "stock": 42,
  "variants": [
    {
      "id": "crew-neck",
      "label": "Crew Neck",
      // no price → uses base €89
      "stock": 42
    },
    {
      "id": "hoodie", "label": "Hoodie",
      "price": 99,   ← overrides base
      "image": "hoodie.jpg",
      "stock": 18
    },
    {
      "id": "zip-hoodie", "label": "Zip Hoodie",
      "price": 109,
      "stock": 0    ← sold-out
    }
  ]
}

// Public helpers:
Shop.variantPrice(product, variantId)
Shop.variantImage(product, variantId)
Shop.variantWeight(product, variantId)
Shop.variantStock(product, variantId)
Shop.variantInStock(product, variantId)

// Add to cart with variant:
Shop.addToCart(product, qty, variantId);

Related Products Strip

Add a related array to any product. A "You may also need" strip appears below the add-to-cart button — auto-wired inside renderShop and renderProductInfo. 

{
  "id": "concrete-desk-lamp",
  "related": [
    {
      "id": "lamp-cable-3m",
      "name": "Textile Cable 3m",
      "price": 18, "weight": 0.2,
      "image": "cable-3m.jpg",
      "description": "Braided cotton."
    },
    { "id": "dimmer", ... },
    { "id": "bulb", ... }
  ]
}

// Manual render:
el.innerHTML = Shop.buildRelatedStrip(
  product, "info"  // "card" | "info"
);
Shop.wireRelatedStrip(el, product);

Back Button (renderBackButton)

Styled back button, injects itself via prepend. Label translates on language change. Pass href, onClick, or neither for history.back().

Live examples (switch language to see labels change)

// Link to page
Shop.renderBackButton("#back-slot", {
  href: "index.html"
});

// Custom static label
Shop.renderBackButton("#back-slot", {
  href: "index.html",
  label: "← Back to overview"
});

// history.back()
Shop.renderBackButton("#back-slot");

// Custom handler
Shop.renderBackButton("#back-slot", {
  onClick: () => myRouter.goBack()
});

Currencies (data/currencies.csv)

All prices stored in EUR. currencies.csv defines rates + buffers. The buffer adds margin on top of the live rate — protecting you if exchange rates drop before settlement.

Integration Guide

Complete setup snippet for any page.

<!-- 1. Styles + scripts -->
<link rel="stylesheet" href="css/shop.css">
<script src="js/config.js"></script>
<script src="js/shop.js"></script>

<!-- 2. Mount points -->
<div id="cart-icon"></div>
<div id="currency"></div>
<div id="shop"></div>

<!-- 3. Language buttons -->
<button class="lumio-lang-btn" data-lang="en">EN</button>
<button class="lumio-lang-btn" data-lang="nl">NL</button>

<script>
Promise.all([
  Shipping.load(),
  Currency.init(),
  Shop.loadLang(),
]).then(() => {
  Shop.wireLanguageSwitcher(".lumio-lang-btn");
  Shop.renderCartIcon({ target: "#cart-icon", fixed: false });
  Shop.renderCurrencySelector("#currency");
  Shop.renderBackButton("#back-slot", { href: "index.html" });

  Shop.renderShop("shop", { showFilter: true });
  Shop.renderProductInfo("product-info", "arc-floor-lamp");
  Shop.renderMiniCart("mini-cart");
});

// Embed from Dornori with lang preset:
// lumio.shop/index.html?lang=nl
</script>
/* Full product JSON schema */
{
  "id": "my-product",
  "name": "My Product",     ← fallback when no i18n
  "price": 89,             ← base EUR price
  "weight": 1.5,           ← kg
  "dimensions": { "l":20, "w":15, "h":40 },
  "image": "main.jpg",
  "images": ["main.jpg", "side.jpg"],
  "url": "",              ← set to make image a link
  "stock": 45, "category": "desk", "featured": false,

  "variants": [           ← optional; overrides price/image/weight/stock
    { "id":"s", "label":"Small",  "price":79,  "stock":20 },
    { "id":"m", "label":"Medium",             "stock":45 },
    { "id":"l", "label":"Large",  "price":89,  "stock":0  }
  ],

  "colors": ["White", "Black"],  ← simple labels, no per-color pricing
  "colors_soldout": ["Black"],  ← visible but disabled

  "related": [            ← accessories below ATC button
    { "id":"cable", "name":"Cable", "price":12, "weight":0.1, "image":"..." }
  ],

  "i18n": {               ← translated name/description/category
    "en": { "name":"My Product",    "description":"...", "category":"desk"    },
    "nl": { "name":"Mijn Product", "description":"...", "category":"bureau"  },
    "no": { "name":"Mitt Produkt", "description":"...", "category":"skrivebord" }
  }
}
