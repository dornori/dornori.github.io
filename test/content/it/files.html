Cosa è incluso

Ogni parte strutturale stampata in 3D per i modelli UFO e Fungo — ottimizzata per la stampa FDM, nessun supporto necessario per la maggior parte delle parti.

  • Corpo base (alloggio motore + batteria)
  • Sezioni tubo × 3 (palo telescopico)
  • Alloggio ingranaggio + staffa motore
  • Testa soucoupe UFO (porta-lampada)
  • Cappello fungo (porta-lampada)
  • Clip guida cavo
  • Staffa morsetto Wago

Impostazioni di stampa

  • Materiale: PLA o PLA+ (PETG funziona anche)
  • Riempimento: 20–30% — giroide o griglia
  • Altezza strato: 0,2mm standard / 0,15mm qualità
  • Supporti: Non richiesti per la maggior parte delle parti
  • Tempo di stampa: ~14 ore totali (tutte le parti)
