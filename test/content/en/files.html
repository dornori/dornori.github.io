What's included

The file pack contains every 3D-printed structural part for both the UFO and Mushroom models — optimised for FDM printing on any standard printer, no supports needed on most parts.

  • Base body (motor + battery housing)
  • Tube sections × 3 (telescoping pole)
  • Gear housing + motor bracket
  • UFO saucer head (lamp holder)
  • Mushroom cap (lamp holder)
  • Cable routing clips
  • Print-in-place hinge for lever switch mount
  • Wago terminal bracket

Print settings (recommended)

MaterialPLA or PLA+ (PETG also works)
Infill20–30% — gyroid or grid
Layer height0.2mm standard / 0.15mm quality
SupportsNone required (most parts)
Print time~14 hours total (all parts)
