Language & Currency

Language priority: ?lang=nl URL param → stored pref → browser Accept-Language → CONFIG.defaultLanguage → "en". Switching fires shop:langChanged — all components re-render without a page reload. Embed from Dornori: webshop.shop/index.html?lang=nl

Live — switch language or currency above ↑

// Boot sequence (always in this order)
Promise.all([
  Shipping.load(),
  Currency.init(),    // IP detect + currencies.csv
  Shop.loadLang(),    // loads ui/{lang}.json + products/{lang}.json
]).then(() => {
  // Wire hidden lang buttons (URL / programmatic still work)
  Shop.wireLanguageSwitcher(".webshop-lang-btn");
  Shop.renderCartIcon({ target: "#cart-slot", fixed: false });
  Shop.renderCurrencySelector("#currency-slot");
  Shop.renderShop("shop", { showFilter: true });
});

// Feature flags in config.js:
CONFIG.features.showLanguageSwitcher = false; // hides buttons, keeps logic
CONFIG.features.showCurrencySelector = true;

// Storage keys (match your wider site):
CONFIG.userPrefs.langKey     = "webshop_lang";

// Language file structure:
// data/lang/ui/en.json        ← UI strings (buttons, labels, messages)
// data/lang/ui/nl.json        ← Dutch UI strings
// data/lang/ui/no.json        ← Norwegian UI strings
// data/lang/products/en.json  ← Product text (name, description, category)
// data/lang/products/nl.json  ← Dutch product text
// data/lang/products/no.json  ← Norwegian product text
//
// Each file contains ONLY human-readable text — safe to send to
// a translator without risking breakage of IDs, prices or config.
CONFIG.userPrefs.currencyKey = "webshop_currency";

Product URL — Clickable Image

Images are not clickable by default. Set product.url to turn the image into an <a> link. Leave it empty (or omit it) and the image is a plain <div> with no pointer cursor.

attachBuyOverlay() was retired in v2. No modal, no overlay. Clean image area.

With url set — pointer cursor

Without url — plain image, no click

// product.url set → image is <a href="...">
{ "url": "https://webshop.shop/products/arc-floor-lamp" }

// product.url empty / absent → image is <div>
{ "url": "" }

// Rendered HTML:
<a href="..." class="webshop-card-img-wrap webshop-card-img-link">   ← url set
<div class="webshop-card-img-wrap">                             ← no url

Variant Pricing

Add a variants array. Each variant can override price, image, weight, stock — any field can be omitted and falls back to the base product value. Selecting a variant updates price, image, and stock state immediately. 

{
  "price": 89,       ← base/fallback price
  "variants": [
    {
      "id": "raw-concrete",
      "label": "Raw Concrete",
      "price": 89, "stock": 23,
      "image": "desk-raw.webp"
    },
    {
      "id": "washed-white",
      "label": "Washed White",
      "price": 95, "stock": 8,
      "image": "desk-white.webp"
    },
    {
      "id": "obsidian-black",
      "label": "Obsidian Black",
      "price": 99,
      "stock": 0   ← sold-out variant
    }
  ]
}

// Public helpers:
Shop.variantPrice(product, variantId)
Shop.variantImage(product, variantId)
Shop.variantWeight(product, variantId)
Shop.variantStock(product, variantId)
Shop.variantInStock(product, variantId)

// Image convention (auto-resolved if no explicit image):
// images/products/{productId}_{variantId}.webp

// Add to cart with variant:
Shop.addToCart(product, qty, "raw-concrete");

Color Swatches & Image Swap

Use colors (string labels, no per-color pricing) when you only need image swapping. Clicking a swatch fades the card image. Image convention: images/products/{productId}_{colorSlug}.webp e.g. globe-pendant_smoke.webp.

Live — click swatches, see image fade

{
  "id": "globe-pendant",
  "image": "globe-pendant.webp",
  "colors": ["Smoke", "Clear", "Amber"]
}

// Auto-resolved image paths:
// globe-pendant_smoke.webp
// globe-pendant_clear.webp
// globe-pendant_amber.webp
// (silent fallback to base image if missing)

// Swatch colors set in shop.css nth-child rules.
// Swatch CSS classes:
// .webshop-color              — base swatch
// .webshop-color--active      — selected
// .webshop-color--soldout     — disabled, strikethrough

Sold-Out Colors & Variants

Sold-out options stay visible (customers know they exist) but are disabled with a diagonal strikethrough on color swatches and struck text on variant buttons. They cannot be selected or added to cart.

Colors — Terracotta sold out

Variants — Obsidian Black sold out

// Sold-out variant:
{ "variants": [
    { "id": "raw-concrete", "stock": 23 },
    { "id": "obsidian-black", "stock": 0 }   ← stock: 0 = sold out
  ]

Related Products Strip

Add a related array to any product. A "You may also need" strip renders below the Add-to-Cart button — automatically inside renderShop and renderProductInfo. Each item has its own add button.

Live — click + to add accessories

{
  "addons": [
    "lamp-cable-3m",
    "dimmer-switch",
    "replacement-bulb-e27"
  ]
}

// addons = IDs of existing products in the catalogue.
// Their name, price, image and description are resolved
// automatically from the product JSON + lang files.
// No duplication of text or prices needed.

// Manual render (auto inside renderProductInfo):
el.innerHTML = Shop.buildRelatedStrip(product, "info");
Shop.wireRelatedStrip(el, product);
// context: "info" = wider layout | "card" = compact

Product Info Widget (renderProductInfo)

Drop a <div id="product-info"> and call Shop.renderProductInfo("product-info", "concrete-desk-lamp"). Gallery, variants, qty, related strip, weight, dimensions — all auto-wired. Rebuilds on shop:langChanged and currency:changed.

Live — Concrete Desk Lamp with 3 variants + related strip

Shop.renderProductInfo("product-info", "concrete-desk-lamp");

// Rebuilds automatically on shop:langChanged + currency:changed

// If product has url set, image wraps in <a href="...">:
{ "url": "https://shop.example.com/products/my-lamp" }

// Image convention (v3):
// Priority: variant.image (JSON) → naming convention → product.image
// Convention: {imageDir}{productId}_{variantId}.{imageExt}
// Config: CONFIG.images.imageDir / CONFIG.images.imageExt

Mini Cart (renderMiniCart)

Live cart summary anywhere on the page. Shows item image (variant-specific), label, weight per line, and totals. Reacts to shop:cartUpdated, shop:langChanged, and currency:changed.

Live — add products above, they appear here

<div id="mini-cart"></div>

Shop.renderMiniCart("mini-cart");

// Listens to these events automatically:
// shop:cartUpdated — cart changed
// shop:langChanged — re-render labels
// currency:changed — re-format prices

// Shows: variant-specific image,
// qty × price, weight per item,
// remove button, shipping/tax/total.

Back Button (renderBackButton)

Styled back button, prepended into any container. Label auto-translates on language change. Pass href, onClick, or neither for history.back().

Live — switch language to see labels change

// Link to page (translated label)
Shop.renderBackButton("#slot", {
  href: "index.html"
});

// Static custom label (not translated)
Shop.renderBackButton("#slot", {
  href: "index.html",
  label: "← Back to overview"
});

// history.back() — no href needed
Shop.renderBackButton("#slot");

// Custom JS handler
Shop.renderBackButton("#slot", {
  onClick: () => myRouter.goBack()
});

Currencies (data/currencies.csv)

All product prices are stored in EUR. The buffer field adds a margin on top of the live rate — protecting you if the exchange rate drops between order and settlement.

Loaded currencies — click any to switch globally

Arc Floor Lamp (€249) in all currencies with buffer

code,symbol,name,rate_to_eur,buffer,decimals,locale
EUR,€,Euro,1.0000,0.0000,2,de-DE
USD,$,US Dollar,1.0870,0.0300,2,en-US    ← buffer = protection margin
GBP,£,British Pound,0.8550,0.0200,2,en-GB
NOK,kr,Norwegian Krone,11.5200,0.2000,0,nb-NO
CZK,Kč,Czech Koruna,25.3000,0.5000,0,cs-CZ

// Buffer math: if real rate is 1.087 and buffer is 0.03,
// you charge at 1.117 — protected if rate drops before settlement.
// €249 × 1.117 = $278.13  ← shown to customer

Integration Guide

Complete copy-paste setup for any page, plus the full product JSON schema.

<!-- 1. Styles + scripts -->
<link rel="stylesheet" href="shop/css/shop.css">
<script src="shop/js/config.js"></script>    <!-- CONFIG + Currency + Shipping + Payment -->
<script src="shop/js/shop.js"></script>

<!-- 2. Mount points -->
<div id="cart-icon"></div>
<div id="currency"></div>
<div id="shop"></div>

<script>
Promise.all([
  Shipping.load(),    // loads data/shipping.csv
  Currency.init(),    // IP detect + data/currencies.csv
  Shop.loadLang(),    // browser detect + data/lang/{lang}.json
]).then(() => {
  Shop.wireLanguageSwitcher(".webshop-lang-btn");
  Shop.renderCartIcon({ target: "#cart-icon", fixed: false });
  Shop.renderCurrencySelector("#currency");
  Shop.renderBackButton("#back-slot", { href: "index.html" });

  Shop.renderShop("shop", { showFilter: true });         // full grid
  Shop.renderProductInfo("product", "arc-floor-lamp");  // single product
  Shop.renderMiniCart("mini-cart");                        // cart sidebar
});

// Payment outcome (dispatched by Payment module):
document.addEventListener("payment:success", e => Shop.clearCart());
document.addEventListener("payment:cancel",  e => Shop.toast("Payment cancelled."));
</script>
/* Full product JSON schema (data/products/my-product.json) */
{
  "id": "my-product",
  "price": 89,                   ← base EUR price
  "weight": 1.5,                 ← kg
  "dimensions": { "l":20, "w":15, "h":40 },
  "image": "main.webp",           ← primary / fallback image
  "images": ["main.webp", "side.webp"],
  "videos": [],
  "url": "",                      ← set to make image a <a> link
  "stock": 45,
  "category": "desk",
  "featured": false,

  "variants": [                  ← optional; overrides price/image/weight/stock
    { "id":"s", "label":"Small",  "price":79,  "stock":20 },
    { "id":"m", "label":"Medium",             "stock":45 },
    { "id":"l", "label":"Large",  "price":89,  "stock":0  }
  ],

  "colors": ["White", "Black"],     ← simple labels, no per-color pricing
  "colors_soldout": ["Black"],    ← visible but unselectable

  "addons": ["cable-id", "dimmer-id"],  ← IDs of accessory products shown in related strip
  "bundled": ["cable-id"],            ← IDs shown in "Frequently Bought Together"
  "bundleDiscount": 10,             ← % off when bundle added together (optional)
  "related": ["other-lamp-id"],      ← IDs of sibling products shown in "You May Also Like"

  "specs": [                     ← optional; rendered as accordion on product page
    { "title": "Features", "title_key": "specs_features",
      "rows": [["Color","Black"],["Material","Steel"]] }
  ]
}

/* Translations live in data/lang/products/{lang}.json — NOT in the product file */
/* data/lang/products/en.json */
{
  "my-product": {
    "name": "My Product",
    "description": "Great product description here.",
    "category": "Desk Lamps"
  }
}
